Ampersand is an experiment.

We took a basic chat client and hooked it up with the Google Translate API. The result? Universal Translator Chat!

Choose any chat room with the field on this page. Once in a room, you can type /languages to get a list of available languages, and choose your language with /language {language name}.

Congratulations, you now speak every language.
